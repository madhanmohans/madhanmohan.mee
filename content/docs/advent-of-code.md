Advent of Code is an annual event featuring Christmas-themed programming challenges that follow an Advent calendar format. It runs from December 1 to December 25 each year, offering daily puzzles that cater to various skill levels and can be solved using any programming language. Participants earn stars for solving problems, with two challenges per day, culminating in a total of 50 stars[1](https://en.wikipedia.org/wiki/Advent_of_Code)[2](https://dev.to/mileswatson/why-you-should-be-doing-advent-of-code-it-s-not-too-late-to-start-3n83)[3](https://dev.to/pnieweglowski/the-benefits-of-advent-of-code-2020-3gic).
## Key Details:

- **Founder**: Eric Wastl
- **First Event**: December 1, 2015
- **Format**: Daily coding puzzles with increasing difficulty.
- **Purpose**: To practice coding skills, engage in problem-solving, and compete on leaderboards.
- **Participation**: Open to developers worldwide, with over 1 million registered users as of 2022[1](https://en.wikipedia.org/wiki/Advent_of_Code)[3](https://dev.to/pnieweglowski/the-benefits-of-advent-of-code-2020-3gic).
