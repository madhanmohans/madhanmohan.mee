# 4 Vs of Big Data

Volume - Variety - Veracity - Velocity

→ **Volume**: scale of data generated
→ **Variety**: different data types/sources
→ **Veracity**: data accuracy and trustworthiness
→ **Velocity**: speed of data generation and processing
