## ==What is Kernel?==

- Kernel is written in C language, compiled with gcc
- Kernel is a low level program in an operating system which performs memory management, process management, task management, disk management.
- There are many kernels available, and we can tweak linux kernels to add/modify features.
- Types
	- Monolithic Kernels
	- MicroKernels
