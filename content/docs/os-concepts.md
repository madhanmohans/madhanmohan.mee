- What is an IP Address?
	IP address stands for **internet protocol address**; it is an identifying number that is associated with a specific computer or computer network. When connected to the internet, the IP address allows the computers to send and receive information.	

- What is MAC Address?
	Medium Access Control Address is used by the routers to identify which address to send the data packet it has recieved from the internet. IP address is the address of a node in the internet using which the data packets end in its desired destination.

- What are different types of protocol?
	-   Transmission Control Protocol (TCP)
	-   Internet Protocol (IP)
	-   User Datagram Protocol (UDP)
	-   Post office Protocol (POP)
	-   Simple mail transport Protocol (SMTP)
	-   File Transfer Protocol (FTP)
	-   Hyper Text Transfer Protocol (HTTP)
	-   Hyper Text Transfer Protocol Secure (HTTPS)
