How to get started at personal level
Create a personal Anthropic account - https://console.anthropic.com/
Purchase credits - $25 is sufficient to get started - https://console.anthropic.com/settings/billing
Create API key and save it somewhere - https://console.anthropic.com/settings/keys
Download and setup Visual Studio Code - https://code.visualstudio.com/
Download and setup Cline with key created above - https://github.com/cline/cline
Experiment on your favourite open source project!
