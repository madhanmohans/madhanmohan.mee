# Generative Adversarial Network

Two competing networks: Generator vs Discriminator

→ Generator creates fake data, Discriminator detects fakes
→ Training through adversarial process
→ [[nvidia-ai-playground]] demos GAN applications
→ Used for image synthesis, style transfer
