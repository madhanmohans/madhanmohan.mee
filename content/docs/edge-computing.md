# edge-computing

Computing at the edge of the network, closer to data sources

Architecture hierarchy:
[[fog-computing]] + [[cloud-computing]]

Example: predictive analytics in General Electric

Uses APIs to get data from [[cloud-computing]]

Enables [[federated-learning]] for privacy
