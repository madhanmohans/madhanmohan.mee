## ==What is soap?:==

### SOAP message has four XML elements:

1. **Envelope** - contains the start and end of the soap message (envelopes everything) 
2. **Header** - specifies application level requirements(authentication status)
3. **Body** - Contains the body of the message(XML requests and response codes)(must)
4. **Fault** - indicate any error messages.(inside body)

Soap bindings use **HTTP** to show *how* the messages are exchanged while soap specs define the **structure** of soap messages.

SOAP is used for [[Remote Procedure calls(RPC)]]
